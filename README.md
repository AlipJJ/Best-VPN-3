## 翻墙软件分为两类
目前最常用的翻墙软件大致分为VPN类和代理类。VPN类包括开源的OpenVPN协议以及商业VPN软件，如ExpressVPN、NordVPN等；代理类翻墙软件包括Shadowsocks、V2Ray等。
